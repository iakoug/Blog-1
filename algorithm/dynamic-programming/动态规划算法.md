# 动态规划算法
动态规划，英文：Dynamic Programming，简称DP，**每一个状态一定是由上一个状态推导出来的**，这一点就区分于贪心，贪心没有状态推导，而是从局部直接选最优的。

## 总结
动态规划的解题步骤/思路：

1. 确定dp数组的下标和值

2. 确定递推公式和初始值（如：当n>2时，递推公式：dp[n] = dp[n-1] + dp[n-2]；当n=1时，dp[1]=1，当n=2时，dp[2]=2）

3. 确定遍历顺序（从前往后/从后往前）

如果发现最终结果不对，首先应该**打印dp数组，排查其输出结果和自己的正确推导结果是否一致**
